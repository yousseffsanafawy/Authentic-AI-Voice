# ============================================================
# Notes on Tiptap v3 (vs v2)
# ============================================================
#
# This project uses Tiptap v3 (installed: 3.22.5).
# Key differences from v2 that affect our code:
#   - @tiptap/pm is now a required explicit peer dependency
#   - Extension API is backwards compatible for StarterKit extensions
#   - Table extensions still work identically
#   - useEditor() and EditorContent API unchanged
#
# ============================================================
# Notes on SSE Streaming (AI Enhancement)
# ============================================================
#
# The useAIEnhance hook (hooks/useAIEnhance.ts) uses the
# native browser fetch() API, NOT Axios, because:
#   - Axios does not support ReadableStream / SSE natively
#   - fetch() + response.body.getReader() is required for SSE
#
